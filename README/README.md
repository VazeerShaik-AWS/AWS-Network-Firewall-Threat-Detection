PROJECT 5

"THREAT HUNTING WITH AWS NETWORK FIREWALL"


OBJECTIVE:

Inspect and control network traffic at VPC level

Detect and block malicious traffic

Centralize network security enforcement


PROBLEM STATEMENT:

Security Groups alone cannot perform deep packet inspection

Limited visibility into network threats


ARCHITECTURE OVERVIEW:

AWS Network Firewall deployed in VPC

Stateful and stateless firewall rules configured

Traffic routed via firewall endpoints

Logs sent to CloudWatch


AWS SERVICES USED:

AWS Network Firewall

Amazon CloudWatch Logs


BUSINESS IMPACT:

Stronger network security posture

Improved compliance and visibility

Reduced attack surface
